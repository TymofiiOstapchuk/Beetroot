Зверстати блок який наданий в figma файлі

Завдання   

1. підключити до файлу  index.html файли
 -  normalize.css (є в каталозі css)
 - свій файл style.css


2. Вимоги
- шрифти мають відповідати шрифтам заданих в макеті

- зображення до блоку підключити як background-image, 
   при цьому висоту цього зображення задавати через background-size: 

- Кнопка з надписом Get started - це <a> 
  і саме цей тег треба через стилі оформити по вигляду як кнопку
