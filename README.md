This space is dedicated to showcasing the homework assignments and projects I am completing during my front-end development course at [Beetroot Academy](https://beetrootacademy.com/). ## 🛠️ Technologies & Skills

Throughout this course, I am mastering the following stack:

* **HTML5:** Writing semantic and accessible page structures.
* **CSS3 / SCSS:** Styling, responsive design, Flexbox, Grid, and preprocessors.
* **JavaScript (ES6+):** Adding interactivity, DOM manipulation, and logic.
* **TypeScript:** Writing scalable, strongly typed code.
* **React:** Building dynamic, component-based user interfaces.
