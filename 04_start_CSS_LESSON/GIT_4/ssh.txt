Перевіряємо саой акаунт (реєструємось) на https://github.com/

=============================== Генерация SSH ключей 

1. Перевіряємо може вже є ключ

---- набираємо команду

ls -al ~/.ssh 

2. Генеруємо SSH ключ

УВАГА !!!!!
Під час генерації ключа SSH буде 2 питання
- шлях де їх зберігати => нічого не набирати, а просто натиснути Enter
- ввести секторну фразу => НІЧОГО НЕ ВВОДИТИ, а просто натиснути ENTER

---- набираємо команду

ssh-keygen -t ed25519 -C "__свій_email__"

3. Додаємо ключ SSH до ssh-agent

---- набираємо команду

eval "$(ssh-agent -s)" 

4. Додаємо ключ на свій GitHub обліковий запис

Потрібно спочатку скопіювати текст публічного ключа

Для цього у Visual Studio Code виконуємо команду

code ~/.ssh/id_ed25519.pub

копіюємо текст ключа, йдемо на свій обліковий запис github

======= На github
- клік за своєю іконкою (праворуч) => у меню вибираємо Settings
- у лівому меню вибираємо пункт SSH and GPS keys
- праворуч буде зелена кнопка New SSH key

- у полі Title даємо назву ключа (будь-яке)
- Key type => Authentication Key
- у полі Key => скопійований текст ключа
- натискаємо кнопку Add SSH key

5. Тестування SSH з'єднання

!!!! УВАГА - на запитання друкуємо yes та натискаємо Enter

---- набираємо команду

ssh -T git@github.com
