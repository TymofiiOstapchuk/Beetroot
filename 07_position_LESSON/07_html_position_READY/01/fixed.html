	<div id="block1">
		<div id="block2">Hello world !!! </div>
	</div>


	.block1 {
		width:350px;
		height:2000px;
		background:gray;
	}
	.block2 {
		width:200px;
		height:200px;
		background:yellow;
   }

Задайте для блоку .block2

	top:200px;
	left:200px;
	position: fixed;

position:fixed - аналогічно властивості position:absolute, але
- відлік координат здійснюється блоку буде від елемента body
- блок не скролитись (прокручуватись) вертикально по екрану
