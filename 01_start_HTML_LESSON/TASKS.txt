--- Для OS Windows
В КОРНІ будь якого диска або на на desktop створити 2 каталога 
 Projects  та _Lessons

--- Для MacOS
Створити такі самі  каталоги  на desktop

В каталог _Lessons копіюємо каталог 01_start_HTML_LESSON
Каталоги з заняттями будуть на кожне заняття
Ці каталоги містять:
- матеріали для занять -> презентації, текстові файли з інформацією
- обов'язково буде zip архів з файлами для занять - наприклад сьогодня це 01_html
